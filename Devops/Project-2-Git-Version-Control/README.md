# Project 2: Version Control with Git

## Objective

Learn and practice basic Git commands and workflows.

---

## Commands Practiced

```bash
git init
git status
git remote -v
git add .
git commit -m "message"
git push origin main
git pull origin main
git clone <repository-url>
git log --oneline
```

---

## Key Learnings

- Repository initialization
- Tracking file changes
- Creating commits
- Synchronizing with GitHub
- Cloning repositories
- Understanding local vs remote repositories

---

## Outcome

Successfully used Git and GitHub to manage project files and maintain version history.
